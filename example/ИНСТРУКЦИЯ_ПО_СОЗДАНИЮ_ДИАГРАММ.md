# Инструкция по созданию диаграмм для курсовой работы

## Варианты создания диаграмм:

### 1. Онлайн инструменты (рекомендуется):

#### Draw.io (https://app.diagrams.net/) - БЕСПЛАТНО
- Откройте сайт https://app.diagrams.net/
- Выберите "Create New Diagram"
- Для ER-диаграмм: File → New → Software → Entity Relationship
- Для IDEF0: Используйте фигуры прямоугольники и стрелки
- Для DFD: File → New → Software → Data Flow Diagram
- Сохраните как PNG/JPG изображение

#### Lucidchart (https://www.lucidchart.com/) - БЕСПЛАТНАЯ регистрация
- Создайте бесплатный аккаунт
- Выберите тип диаграммы (ER Diagram, Flowchart для DFD)
- Создайте диаграмму
- Экспортируйте как PNG/JPG

#### dbdiagram.io (https://dbdiagram.io/) - для ER-диаграмм БД
- Откройте сайт
- Введите текст описания отношений
- Экспортируйте как PNG

### 2. Desktop приложения:

#### yEd Graph Editor (https://www.yworks.com/products/yed) - БЕСПЛАТНО
- Скачайте и установите
- Создайте новую диаграмму
- Используйте готовые шаблоны для ER, DFD
- Экспортируйте как PNG/JPG

#### Microsoft Visio (платный, если есть доступ через вуз)

### 3. Экспорт из Mermaid (если используете Markdown редактор с поддержкой Mermaid):

Если ваш Markdown редактор поддерживает Mermaid (VS Code с расширением, GitHub, GitLab):
1. Диаграммы уже добавлены в документ в формате Mermaid
2. Используйте онлайн конвертер: https://mermaid.live/ 
3. Вставьте код диаграммы
4. Экспортируйте как PNG/SVG

### 4. Скриншоты из онлайн редакторов Mermaid:

1. Перейдите на https://mermaid.live/
2. Скопируйте код диаграммы из документа (начинается с ```mermaid)
3. Вставьте в редактор
4. Нажмите "Actions" → "Download PNG" или "Download SVG"

## Что нужно создать:

### ER-диаграммы (5 штук):
- Рисунок 1.1 – ER-диаграмма "Товар принадлежит бренду"
- Рисунок 1.2 – ER-диаграмма "Товар принадлежит категории"
- Рисунок 1.3 – ER-диаграмма "Пользователь оформляет заказ"
- Рисунок 1.4 – ER-диаграмма "Заказ состоит из элементов заказа"
- Рисунок 1.5 – ER-диаграмма "Элемент заказа содержит товар"

### IDEF0 диаграмма:
- Диаграмма IDEF0 верхнего уровня (A0): "Управление интернет-магазином брендовой одежды"

### DFD диаграмма:
- Диаграмма DFD верхнего уровня

## Структура ER-диаграмм:

Каждая ER-диаграмма содержит:
- Две сущности (прямоугольники)
- Связь между ними (ромб или линия с названием)
- Кардинальность связи (1:M, M:1, 1:1)
- Класс принадлежности (обязательный/необязательный - отображается как жирная/обычная линия или специальными символами)

## Экспорт диаграмм Mermaid (уже добавлены в документ):

Если в документе уже есть диаграммы в формате Mermaid (код между ```mermaid и ```):

1. **Способ 1 - Mermaid Live Editor (самый простой):**
   - Перейдите на https://mermaid.live/
   - Скопируйте код диаграммы из документа (все между ```mermaid и ```)
   - Вставьте в редактор слева
   - Диаграмма отобразится справа
   - Нажмите "Actions" → "Download PNG" или "Download SVG"
   - Сохраните изображение

2. **Способ 2 - VS Code:**
   - Откройте файл .md в VS Code
   - Установите расширение "Markdown Preview Mermaid Support"
   - Откройте Preview (Ctrl+Shift+V)
   - Кликните правой кнопкой на диаграмме → "Copy Image" или используйте расширение для экспорта

3. **Способ 3 - GitHub:**
   - Загрузите файл на GitHub
   - GitHub автоматически отрендерит Mermaid диаграммы
   - Сделайте скриншот или используйте инструменты разработчика для сохранения изображения

## Рекомендации:

1. **Для быстрого результата:** Используйте Mermaid Live Editor для экспорта уже добавленных диаграмм
2. **Для профессионального вида:** Используйте **Draw.io** для создания диаграмм с нуля
3. Создавайте диаграммы в высоком разрешении (минимум 300 DPI для печати)
4. Сохраняйте в формате PNG (для Word) или JPG
5. Вставляйте изображения в Word и подписывайте их как "Рисунок X.X"
6. Для IDEF0 лучше использовать Draw.io или специальные инструменты (Mermaid не поддерживает стандартный формат IDEF0)
