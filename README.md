# Онлайн-магазин брендовой одежды

Веб-приложение на ASP.NET Blazor Server для продажи брендовой одежды.

## Технологии

- .NET 8
- ASP.NET Core Blazor Server
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- EPPlus (для работы с Excel)

## Функциональность

- Каталог товаров с фильтрацией
- Корзина покупок
- Оформление заказов
- Система авторизации с ролями (Admin, Manager, Customer)
- CRUD операции для товаров, брендов, категорий
- Экспорт/импорт данных в Excel
- Личный кабинет пользователя

## Установка и запуск

1. Убедитесь, что установлен .NET 8 SDK
2. Убедитесь, что установлен SQL Server (или SQL Server LocalDB)
3. Обновите строку подключения в `appsettings.json`
4. Выполните миграции базы данных (автоматически при первом запуске)
5. Запустите приложение

- `Models/` - Модели данных
- `Data/` - DbContext и инициализация данных
- `Services/` - Бизнес-логика
- `Components/Pages/` - Blazor компоненты страниц
