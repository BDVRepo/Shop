# Работа с изображениями товаров

## Формат именования файлов

Все изображения товаров должны быть размещены в папке `wwwroot/images/products/` и именоваться по следующему формату:

```
{id}.jpg
```

Где `{id}` - это ID товара в базе данных.

### Примеры:
- Товар с ID = 1 → файл `1.jpg`
- Товар с ID = 2 → файл `2.jpg`
- Товар с ID = 123 → файл `123.jpg`

## Рекомендуемые размеры изображений

### Универсальный вариант (рекомендуется):
- **Размер**: **1000x1000px** (квадрат)
- **Формат**: JPG (качество 85-90%)
- **Размер файла**: 100-200 KB

**Преимущества**:
- Подходит для всех мест отображения
- Хорошее качество на Retina дисплеях
- Не слишком большой размер файла
- Квадратная форма хорошо обрезается с помощью `object-fit: cover`

### Размеры отображения в приложении:

1. **Главная страница (карточки товаров)**: 
   - Высота: 250px
   - Ширина: адаптивная (~300-400px на десктопе)
   - Используется: `object-fit: cover` (изображение обрезается по центру)

2. **Страница каталога (карточки товаров)**:
   - Высота: 200px
   - Ширина: адаптивная (~300-400px на десктопе)
   - Используется: `object-fit: cover`

3. **Страница детального просмотра товара**:
   - Ширина: адаптивная, до ~600px на десктопе (col-md-6)
   - Высота: автоматическая, сохраняет пропорции
   - Используется: `img-fluid` (адаптивное масштабирование)

## Требования к изображениям

1. **Соотношение сторон**: Квадратные (1:1) или близкие к квадрату (4:3, 3:4)
2. **Формат**: JPG только
3. **Качество**: 85-90% для JPG (баланс качества и размера)
4. **Цветовое пространство**: RGB, sRGB
5. **Максимальный размер файла**: до 500 KB (для быстрой загрузки)

## Добавление изображений товаров

1. Создайте товар в админ-панели (при создании товару будет присвоен ID)
2. Найдите ID товара в списке товаров или базе данных
3. Подготовьте изображение в формате JPG размером 1000x1000px
4. Сохраните изображение в папку `wwwroot/images/products/` с именем `{id}.jpg`
5. Изображение автоматически отобразится на всех страницах товара

## Placeholder изображение

Если изображение товара отсутствует или не загружено, используется placeholder:
- Путь: `/images/products/placeholder.svg`
- Автоматически подставляется браузером при ошибке загрузки изображения

## Важные замечания

- **Имя файла должно точно соответствовать ID товара**: `1.jpg`, `2.jpg` и т.д.
- **Используется только формат JPG**
- **Изображения не хранятся в базе данных**, они являются статическими файлами
- После удаления товара рекомендуется также удалить его изображение из папки `wwwroot/images/products/`
